\doxysubsection{Detailed Description}
Writes and reads a \doxylink{class_data_file_header}{Data\+File\+Header} as the first record of a .li file. 

Responsibilities:\+
\begin{DoxyItemize}
\item \doxylink{class_header_buffer_ab3b88e0504865d9ec587056f07ea9c09}{write\+Header()} -- packs the header and writes it with a 2-\/byte length prefix to byte 0 of an open output stream
\item \doxylink{class_header_buffer_a2e36ff2de2b91057f336b48be41a2753}{read\+Header()} -- reads the 2-\/byte length, reads the payload, and calls \doxylink{class_data_file_header_aa490a0d7ef821d0837a3abe72726fc3f}{Data\+File\+Header::\+unpack()} to restore all fields
\end{DoxyItemize}

NOT responsible for:\+
\begin{DoxyItemize}
\item Opening or closing files (caller does that)
\item Storing the \doxylink{class_data_file_header}{Data\+File\+Header} (caller owns it)
\end{DoxyItemize}

\begin{DoxyPrecond}{Precondition}
Streams passed to read/\+write methods must already be open in binary mode and positioned at byte 0. 
\end{DoxyPrecond}
\begin{DoxyPostcond}{Postcondition}
Stream position advances past the header record after each call. 
\end{DoxyPostcond}
