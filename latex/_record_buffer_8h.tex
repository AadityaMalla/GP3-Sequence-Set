This class is responsible for converting a \doxylink{struct_zip_code_record}{Zip\+Code\+Record} to and from a raw byte representation stored inside a block.

GP 3.\+0 still expects the record fields to remain comma-\/separated in logical form, so this buffer stores each record as:\+

\+[2-\/byte big-\/endian payload length]\+ \+[ASCII payload:\+ ZZZZZ,place,state,county,lat,lon]\+

This allows records to remain variable-\/length while still being easy to pack into a fixed-\/size block.
