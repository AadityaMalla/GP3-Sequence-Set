\doxysubsection{Detailed Description}
Stores metadata for a blocked sequence-\/set file. 

Responsibilities:\+
\begin{DoxyItemize}
\item store file-\/level metadata
\item store field descriptions
\item pack/\+unpack itself into a raw buffer
\item display itself for debugging
\end{DoxyItemize}

NOT responsible for:\+
\begin{DoxyItemize}
\item opening or closing files
\item reading/\+writing streams directly 
\end{DoxyItemize}
