\doxysubsection{Detailed Description}
Converts \doxylink{struct_zip_code_record}{Zip\+Code\+Record} objects to/\+from raw bytes for block storage. 

Responsibilities:\+
\begin{DoxyItemize}
\item compute packed size of one record
\item pack one record into a caller-\/provided buffer
\item unpack one record from a raw buffer
\item encode/\+decode the 2-\/byte big-\/endian length prefix
\end{DoxyItemize}

NOT responsible for:\+
\begin{DoxyItemize}
\item file I/\+O
\item whole-\/block management
\item indexing 
\end{DoxyItemize}
