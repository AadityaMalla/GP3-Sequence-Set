\doxysubsection{Detailed Description}
Data structure for the header record of a length-\/indicated ZIP code file. 

\begin{DoxyAuthor}{Author}
CSCI 331 Team 2 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
03/\+13/\+2026
\end{DoxyDate}
This struct holds all metadata about a .li data file. It knows how to pack/\+unpack itself into/\+from a raw byte buffer, but has NO file I/\+O — that responsibility belongs to \doxylink{class_header_buffer}{Header\+Buffer}.

The header is always stored as the FIRST length-\/indicated record inside the .li file (byte 0), using the same 2-\/byte length prefix format as every other record.
