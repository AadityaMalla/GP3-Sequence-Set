This class manages the physical blocked file on disk.

File layout:\+ \+[2-\/byte header length]\+\+[header payload]\+ \+[block 0 bytes]\+ \+[block 1 bytes]\+ \+[block 2 bytes]\+ ...

Responsibilities:\+
\begin{DoxyItemize}
\item open/\+close the file
\item write/\+read the blocked file header
\item write/\+read one fixed-\/size block by RBN
\item compute byte offsets for blocks
\end{DoxyItemize}

NOT responsible for:\+
\begin{DoxyItemize}
\item sequence-\/set logic
\item block splitting/\+merging
\item indexing 
\end{DoxyItemize}
