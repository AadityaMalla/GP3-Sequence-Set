\doxysubsection{Detailed Description}
In-\/memory representation of one physical block. 

Responsibilities:\+
\begin{DoxyItemize}
\item store active-\/block metadata
\item store avail-\/block metadata
\item keep records sorted by ZIP code
\item pack/\+unpack the full fixed-\/size block image
\item calculate used/\+free bytes
\end{DoxyItemize}

NOT responsible for:\+
\begin{DoxyItemize}
\item file I/\+O
\item global sequence-\/set management
\item index creation 
\end{DoxyItemize}
