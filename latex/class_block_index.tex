\doxysubsection{Detailed Description}
In-\/memory simple index for the blocked sequence-\/set file. 

Responsibilities:\+
\begin{DoxyItemize}
\item store block index entries
\item sort entries
\item write/\+read binary index file
\item find candidate block for a ZIP code via binary search 
\end{DoxyItemize}
