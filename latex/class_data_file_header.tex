\doxysubsection{Detailed Description}
Holds metadata about a length-\/indicated ZIP code data file. 

Responsibilities:\+
\begin{DoxyItemize}
\item Store file-\/level metadata (type, version, record count, etc.)
\item Describe each field via a \doxylink{struct_field_descriptor}{Field\+Descriptor} vector
\item Pack itself into a raw byte buffer (used by \doxylink{class_header_buffer}{Header\+Buffer} to write)
\item Unpack itself from a raw byte buffer (used by \doxylink{class_header_buffer}{Header\+Buffer} to read)
\item Display its contents for debugging
\end{DoxyItemize}

NOT responsible for:\+
\begin{DoxyItemize}
\item Opening or closing files (\doxylink{class_header_buffer}{Header\+Buffer} does that)
\item Writing or reading from streams (\doxylink{class_header_buffer}{Header\+Buffer} does that)
\end{DoxyItemize}

\begin{DoxyPrecond}{Precondition}
All string fields must not require binary-\/safe escaping (no embedded nulls). 
\end{DoxyPrecond}
\begin{DoxyPostcond}{Postcondition}
An initialised \doxylink{class_data_file_header}{Data\+File\+Header} can be packed into a buffer and written by \doxylink{class_header_buffer}{Header\+Buffer} as the first record of a .li file. 
\end{DoxyPostcond}
