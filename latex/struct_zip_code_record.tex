\doxysubsection{Detailed Description}
Structure to hold information for a single ZIP code This structure contains all fields from the US Postal Code database:\+ 


\begin{DoxyItemize}
\item ZIP code (5-\/digit postal code)
\item Place name (city/\+town name)
\item State (2-\/character state abbreviation)
\item County name
\item Geographic coordinates (latitude and longitude) Longitude values are negative for locations west of the Prime Meridian, and latitude values are positive for locations north of the Equator. \begin{DoxyPrecond}{Precondition}
All fields must be properly initialized before use 
\end{DoxyPrecond}
\begin{DoxyPostcond}{Postcondition}
Instances of \doxylink{struct_zip_code_record}{Zip\+Code\+Record} can be used to store and retrieve ZIP code information 
\end{DoxyPostcond}
\begin{DoxyNote}{Note}
This structure serves as the fundamental data unit for managing and processing ZIP code information in our application. 
\end{DoxyNote}

\end{DoxyItemize}
